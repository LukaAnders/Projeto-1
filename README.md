### Instruções de uso


1. Salve cada bloco em um arquivo com o nome indicado: `index.html`, `styles.css` e `script.js`.
2. Coloque todos os arquivos na mesma pasta.
3. Abra `index.html` no navegador.


Se quiser que eu gere um ZIP com os três arquivos ou que eu adicione funcionalidades (carrinho, busca, filtros, modal de detalhes), posso fazer — diga qual funcionalidade deseja em seguida.
